# Notebook Web App

A full-stack note-taking application with secure authentication.

## Tech Stack
- Frontend: React.js
- Backend: Spring Boot
- Database: MySQL
- Authentication: JWT

## Features
- User authentication (JWT)
- Create, update, delete notes
- Secure user-specific data

## Project Structure
frontend/ - React application  
backend/ - Spring Boot API  

## How to Run

Frontend:
npm install
npm start

Backend:
mvn spring-boot:run
